# 遗韵养生 - 非遗与健康微信小程序

本项目是一款结合“非物质文化遗产（非遗）”与“健康养生”为主题的微信小程序。整体设计采用“清新古风”，以自然舒适的松绿、浅青色为主色调，为用户提供沉浸式的非遗健康文化体验。

## 🎯 核心功能板块

1. **注册与登录体系**
   - 支持手机号+密码登录。
   - 提供注册功能。
   - 支持通过验证码找回并重置密码。
   
2. **首页（非遗文化大厅）**
   - **轮播展示**：展示最新非遗与健康结合的活动宣传。
   - **四大分类**：
     - **传统体育**：如太极拳、八段锦等强身健体功法。
     - **传统饮食**：如药膳同源、非遗茶艺、传统节气饮食等。
     - **传统医药**：如针灸、推拿、中医药理等。
     - **传统音乐**：如古琴、五音疗疾等音乐疗愈内容。
   - **活动与资讯**：提供非遗文化教学与科普文章展示。

3. **AI 问答**
   - 接入 AI 大模型对话界面，解答用户关于非遗和健康的各类疑问。
   - 预设“猜您想问”推荐问题（如：“什么是八段锦？”、“秋季适合吃哪些非遗美食？”），方便用户快速发起对话。
   
4. **个人中心**
   - 展示用户基础信息（头像、手机号等）。
   - 提供**留言与解读**功能入口。
   - 提供**帮助与反馈**功能入口。
   - 支持**切换账号**与**退出登录**。

## 🎨 UI 设计规范
- **风格**：清新古风、极简优美。
- **主色调**：
  - 深松绿 (`#2E5A42`)：用于标题、按钮及强调元素。
  - 浅青/豆绿 (`#F2F6F0`, `#E8F4EC`)：用于背景、卡片底色及分割线。
- **字体**：以圆润、清晰的无衬线字体为主，辅以部分传统衬线体（如分类图标）提升古风韵味。

## 🚀 运行说明

1. 请下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
2. 在微信开发者工具中，选择“导入项目”。
3. 目录选择当前的 `feiyi+jiankang` 文件夹。
4. AppID 可选择“测试号”。
5. 导入成功后，即可在模拟器中预览“清新古风”的非遗健康小程序！

## 📂 目录结构

```text
├── app.js               # 全局逻辑（包含登录态管理）
├── app.json             # 全局配置（页面路由、底部 TabBar、顶部导航栏）
├── app.wxss             # 全局样式（古风颜色变量、公共类）
├── project.config.json  # 小程序项目配置
├── pages/
│   ├── login/           # 登录、注册、找回密码页
│   ├── index/           # 首页（四大分类、宣传内容）
│   ├── ai/              # AI 问答对话页
│   └── profile/         # 个人中心（用户管理、留言反馈）
└── images/              # 存放 TabBar 及其他本地图片资源 (需自行放入设计好的图标)
```

## 📝 后续开发建议

- **图片资源**：请在 `images/` 目录下放置古风风格的 TabBar 图标（`home.png`, `ai.png`, `profile.png` 及对应 `*-active.png`）。
- **AI 接口对接**：在 `pages/ai/ai.js` 中将模拟的定时器回复替换为真实的 AI 接口（如文心一言、通义千问等大模型 API）调用。
- **后台数据对接**：将首页的非遗文章列表、分类跳转与您的服务器数据库进行连接。
